--- Gerenciador de Tarefas ---

Um projeto simples de gerenciador de tarefas rodando no terminal, desenvolvido em Python como parte da minha jornada de aprendizado.

--- O que o projeto faz ---

O usuário consegue adicionar, listar, concluir e remover tarefas direto pelo terminal. As tarefas ficam salvas num arquivo JSON, então nada se
perde quando o programa é fechado.

--- O que foi usado ---

- Python — listas, loops, condicionais, input/output
- JSON — primeira vez trabalhando com persistência de dados em arquivo
- os — para verificar se o arquivo de dados já existe antes de carregar

--- O que ainda falta ---

Esse projeto ainda vai crescer bastante! Os planos são:
- Adicionar prioridades e datas nas tarefas
- Criar uma API com Flask ou FastAPI
- Desenvolver uma interface web com HTML, CSS e JavaScript
